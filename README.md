# http-servers

## Project Overview 
A simple HTTP server implemented in C that can handle text and image responses.

The project includes: 
  A simple HTTP server that handles one client connection at a time
	A multithreaded HTTP server that handles multiple clients simultaneously. 

Features:
	A multithreaded server that creates a thread pool to handle multiple clients
	Direct communication between client and server for a single-threaded server
	Supports HTTP requests and responses between clients and the server
  
The skills learned:
  TCP socket programming in C
  Server-side communication for TCP
  HTTP requests and response parsing 
  Signal handling for server termination 
  Robust error checking for socket operation, file I/O, and thread management

Collaboration:
	This project was completed in collaboration with another student. Some starter code and scripts were provided by instructors from the course.
